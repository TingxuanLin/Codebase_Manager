# Codebase Manager Frontend

React and TypeScript frontend for Codebase Manager, built with Vite.

## Requirements

- Node.js 20 or newer
- npm

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

The app starts on:

```text
http://localhost:5173
```

Vite proxies `/api/*` requests to the backend at `http://localhost:8080`.

## Build

```bash
npm run build
```
